# Snapflect Assessment Portal: RBAC Menu Matrix (Version 1.0)

This document serves as the official reference for the Role-Based Access Control (RBAC) navigation matrix implemented in the frontend application. 

It defines the exact menu items visible to each of the 10 core personas when they log into the system.

## Notes for Testing

**Local Development Bypass:** 
For rapid UI/UX testing locally, you do not need to enter passwords. The backend includes a `MockAuthMiddleware` which intercepts the `mock_email` query parameter.

To switch personas on the fly, append the parameter to the Angular dashboard URL:
`http://localhost:4200/dashboard?mock_email=candidate@snapflect.com`

**Default Passwords:** 
If testing via the actual `/auth/login` page, the default password generated by `UserSeeder.php` for all accounts is:
`ChangeMeImmediately`
