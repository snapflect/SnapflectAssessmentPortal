This document defines the mandatory development standards for:
Folder StructureNamespacesCoding StandardsNaming ConventionsServicesRepositoriesDTOsControllersPoliciesEventsJobsTestingException Handling
All generated code must comply with this document.

Principle 1
Single Responsibility
One ClassOne Responsibility

Principle 2
Dependency Inversion
Depend On InterfacesNot Implementations

Principle 3
Separation Of Concerns
ControllersServicesRepositoriesPoliciesDTOsMust Remain Separate

Principle 4
Modular Design
Each Business DomainOwn Module
