INTEGRATION_EVENTS
Purpose
Tracks system-to-system integration activities.
Supports:
API CallsWebhook ProcessingImport OperationsExport OperationsSystem Integrations

Governance
Integration events are operational records.
They are not replacements for audit records.

CREATETABLE integration_events(id BIGINT UNSIGNED NOTNULL AUTO_INCREMENT,    uuid CHAR(36) NOTNULL,    correlation_id CHAR(36) NULL,    event_code VARCHAR(100) NOTNULL,    event_category VARCHAR(50) NOTNULL,    event_status VARCHAR(50)NOTNULLDEFAULT'Pending',    event_payload_json JSON NOTNULL,    metadata_json JSON NULL,    event_date DATETIME(3)NOTNULLDEFAULTCURRENT_TIMESTAMP(3),PRIMARYKEY (id),CONSTRAINT ux_integration_events_uuidUNIQUE (uuid))ENGINE=InnoDBDEFAULT CHARSET=utf8mb4COLLATE=utf8mb4_unicode_ci;

AUDIT_EVENTS
Purpose
Stores immutable enterprise audit records.
Supports:
Security AuditingCompliance AuditingChange TrackingForensic InvestigationRegulatory Reporting

Governance
Audit records are append-only.
Audit records must never be updated or deleted through normal application workflows.

CREATETABLE audit_events(id BIGINT UNSIGNED NOTNULL AUTO_INCREMENT,    uuid CHAR(36) NOTNULL,    audit_category VARCHAR(50) NOTNULL,    event_type VARCHAR(100) NOTNULL,    entity_type VARCHAR(100) NOTNULL,    entity_id BIGINT UNSIGNED NULL,    performed_by BIGINT UNSIGNED NULL,    source_type VARCHAR(50) NOTNULL,    before_state_json JSON NULL,    after_state_json JSON NULL,    metadata_json JSON NULL,    retention_expiry_date DATETIME(3) NULL,    event_date DATETIME(3)NOTNULLDEFAULTCURRENT_TIMESTAMP(3),PRIMARYKEY (id),CONSTRAINT ux_audit_events_uuidUNIQUE (uuid))ENGINE=InnoDBDEFAULT CHARSET=utf8mb4COLLATE=utf8mb4_unicode_ci;
