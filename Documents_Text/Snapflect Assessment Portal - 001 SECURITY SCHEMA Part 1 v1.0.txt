BUSINESS RULES
USERS
Email must be unique.
Mobile number must be unique when provided.
Passwords must be stored using BCrypt.
Soft delete only.
Deleted users are never physically removed.
Email verification required for self-registration.
Five failed login attempts trigger account lock.
Lock duration: 30 minutes.

USER_PROFILES
One profile per user.
Profile created during onboarding.
User may update profile anytime.
Profile deletion follows parent user lifecycle.
