PROCTORING_SESSIONS
Purpose
Tracks the complete proctoring lifecycle for an assessment attempt.
Examples:
Assessment StartedCamera ActivatedMicrophone ActivatedScreen Sharing ActivatedAssessment Submitted

Governance
One assessment attempt must have exactly one proctoring session.

CREATETABLE proctoring_sessions(id BIGINT UNSIGNED NOTNULL AUTO_INCREMENT,    uuid CHAR(36) NOTNULL,    assessment_attempt_id BIGINT UNSIGNED NOTNULL,    proctoring_mode VARCHAR(50)NOTNULLDEFAULT'AI',    session_status VARCHAR(50)NOTNULLDEFAULT'Pending',    camera_enabled TINYINT(1)NOTNULLDEFAULT0,    microphone_enabled TINYINT(1)NOTNULLDEFAULT0,    screen_share_enabled TINYINT(1)NOTNULLDEFAULT0,    integrity_score DECIMAL(5,2)NULL,    metadata_json JSON NULL,    started_date DATETIME(3) NULL,    ended_date DATETIME(3) NULL,    created_date DATETIME(3)NOTNULLDEFAULTCURRENT_TIMESTAMP(3),PRIMARYKEY (id),CONSTRAINT ux_proctoring_sessions_uuidUNIQUE (uuid),CONSTRAINT ux_proctoring_session_attemptUNIQUE (assessment_attempt_id))ENGINE=InnoDBDEFAULT CHARSET=utf8mb4COLLATE=utf8mb4_unicode_ci;

PROCTORING_SESSIONS FOREIGN KEYS
ALTERTABLE proctoring_sessionsADDCONSTRAINT fk_proctoring_sessions_attemptFOREIGNKEY (assessment_attempt_id)REFERENCES assessment_attempts(id);

PROCTORING_VIOLATIONS
Purpose
Stores suspected assessment integrity violations.
Examples:
Multiple Persons DetectedCamera DisabledBrowser SwitchVirtual Machine DetectedUnauthorized Device

Governance
Violations may be created by AI, reviewers, or system policies.

CREATETABLE proctoring_violations(id BIGINT UNSIGNED NOTNULL AUTO_INCREMENT,    uuid CHAR(36) NOTNULL,    proctoring_session_id BIGINT UNSIGNED NOTNULL,    correlation_id CHAR(36) NULL,    violation_type VARCHAR(100) NOTNULL,    risk_score DECIMAL(5,2)NOTNULL,    status VARCHAR(50)NOTNULLDEFAULT'Open',    detected_by VARCHAR(50)NOTNULLDEFAULT'AI',    evidence_count INTNOTNULLDEFAULT0,    metadata_json JSON NULL,    detected_date DATETIME(3)NOTNULLDEFAULTCURRENT_TIMESTAMP(3),PRIMARYKEY (id),CONSTRAINT ux_proctoring_violations_uuidUNIQUE (uuid))ENGINE=InnoDBDEFAULT CHARSET=utf8mb4COLLATE=utf8mb4_unicode_ci;
