2. SCHEMA ENHANCEMENT
2.1 Question Option Randomization Support
Reason
The current architecture supports:
Question Bank Randomization
through:
question_bank_versions.randomize_question_order
However many assessment engines require:
Question remains fixedOptions are randomized
Example:
Version A
A. InterfaceB. ClassC. MethodD. Delegate
Version B
C. MethodA. InterfaceD. DelegateB. Class
without modifying the Question itself.

Approved Enhancement
Table
question_versions
New Column
randomize_option_order TINYINT(1)NOTNULLDEFAULT0

Column Meaning
0 = Fixed Option Order1 = Randomized Option Order

Runtime Behavior
When:
randomize_option_order = 1
Runtime engine shall randomize display order of options.
When:
randomize_option_order = 0
Options shall be displayed using:
question_options.display_order

Updated Question Version Definition
The approved logical structure becomes:
question_versionsquestion_typedifficulty_levelmarksnegative_marksestimated_duration_minutesis_randomizablerandomize_option_ordermetadata_json

3. GOVERNANCE ENHANCEMENTS
3.1 Negative Mark Validation Rule
Rule
negative_marks <= marks
must always be true.

Invalid Example
Marks = 1Negative Marks = 2

Valid Example
Marks = 2Negative Marks = 0.50

Enforcement
Application Validation Layer

3.2 Single Choice Validation Rule
Applies To
question_type = SingleChoice

Rule
Exactly one answer must be marked:
is_correct = 1

Valid Example
A = CorrectB = IncorrectC = IncorrectD = Incorrect

Invalid Example
A = CorrectC = Correct

Enforcement
Application Validation Layer

3.3 Multiple Choice Validation Rule
Applies To
question_type = MultipleChoice

Rule
At least one correct answer must exist.

Valid Example
A = CorrectC = CorrectD = Correct

Invalid Example
No Correct Answers

Enforcement
Application Validation Layer

3.4 Random Selection Validation Rule
Applies To
question_bank_versions.selection_strategy=RandomSelection

Rule
question_count > 0
must be true.

Additional Rule
question_count<=available active questions
must be true.

Invalid Example
Question Count = 20Available Questions = 10

Enforcement
Application Validation Layer
