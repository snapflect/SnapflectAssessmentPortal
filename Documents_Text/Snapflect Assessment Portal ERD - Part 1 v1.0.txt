12. SECURITY RULES
Passwords shall never be stored in plain text.
Passwords shall be stored using bcrypt.
JWT tokens shall never be stored in plain text.
Sensitive tokens should be hashed before persistence.
Email verification tokens shall expire.
Password reset tokens shall expire.
Soft delete shall be enforced.
