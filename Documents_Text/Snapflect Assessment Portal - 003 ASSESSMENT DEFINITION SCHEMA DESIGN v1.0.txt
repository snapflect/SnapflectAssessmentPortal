1. PURPOSE
The Assessment Definition Schema is the core content management layer of the Assessment Portal.
It defines:
Technologies
Assessments
Sections
Question Banks
Questions
Question Options
Question Answers
Content Versioning
This module is responsible for creating and managing assessment content.

2. CONTENT HIERARCHY
Logical User Hierarchy
Technology    ↓Assessment    ↓Section    ↓Question Bank    ↓Question

Physical Database Architecture
Technology    ↓Assessment    ↓SectionQuestion Bank    ↓QuestionSection    ↓Section Question Banks    ↓Question Bank
This architecture allows Question Bank reuse across multiple sections.
