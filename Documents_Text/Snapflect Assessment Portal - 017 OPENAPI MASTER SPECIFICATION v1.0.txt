1. PURPOSE
This document establishes the master OpenAPI governance framework for Snapflect.
It defines:
API StandardsOpenAPI StandardsContract StandardsDTO StandardsResponse StandardsError StandardsSecurity StandardsDocumentation Standards
This document is the authoritative source for all API contracts.

2. OPENAPI VERSION
Approved Standard:
openapi:3.1.0
All future API specifications shall comply with OpenAPI 3.1.

3. API VERSIONING STRATEGY
Current Version:
v1
Base URL:
/api/v1
Examples:
/api/v1/auth/login/api/v1/users/api/v1/assessments/api/v1/questions

Breaking Changes
Breaking changes require:
New API Version
Example:
v1 → v2

Non-Breaking Changes
Allowed:
Optional FieldsAdditional PropertiesNew Endpoints
