5. EVALUATION GOVERNANCE ENHANCEMENT
Evaluation Completion Rule
When:
evaluation_status = Completed
Then:
evaluated_date
must be populated.

Failed Evaluation Rule
When:
evaluation_status = Failed
Then:
result_status
must remain:
Pending
until reprocessing succeeds.

6. CLIENT METADATA GOVERNANCE
Client metadata is informational only.
The platform must never:
Authorize AccessGrant PermissionsDetermine Security Roles
based solely on values stored in:
client_metadata_json

7. RESPONSE METADATA GOVERNANCE
Response metadata must never be treated as the authoritative answer.
Authoritative answer remains:
response_value
Metadata exists only to support:
EvaluationAnalyticsQuestion-Type ExtensionsAI Services
